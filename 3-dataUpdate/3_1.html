Layer’s Styles

Layer’s Styles#

We focus on customising the WQP map view, information and management on this page.

The first step is entering the layer page that will be edited, and you achieve this by clicking on the layer’s name. Fig. 13 presents the view of the example layer page.

To start editing a layer, click on the Editing Tools button and select the Styles Manage option. At this point, you can pick between the different styles available in the platform to apply to the layer.

Fig. 15 presents the default for the layer. However, to select a new layer style, you can open the dropdown menu to pick an alternative style. Then, click on Update Styles to add the updated styles to the layer.

For the WQPs maps, a set of styles have been uploaded for their visualization.

Notice that the style hasn’t been applied to the layer yet. Now, the user must predefine the default style for the layer. Go back to the Styles Page of the layer (see Fig. 15) and set the one corresponding to the map as the default style.

The layer’s style will now match the one defined in the previous steps. Verify that this is correct by reviewing the corresponding layer page.

Alternatively, you can upload a different style to apply to the layer by selecting the upload option in the Upload option in the Editing Instruments menu (see Fig. 14).

An additional step for customising the view of a layer in the Layer is to edit the thumbnail accompanying the Layer in the list (e.g. setting an image or a zoomed view of the Layer). To edit the thumbnail, access the Layer Page and zoom in to the area of interest using the map view on the page. After doing so, select Editing Tools and click Set. Once the platform prompts the successful image change (see Fig. 20), click OK and go back to the Layers Page to view the updated image (see Fig. 21).
