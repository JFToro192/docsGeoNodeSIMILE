SIMILE'S Collaborative Data Sharing Platform
============================================

The platform for sharing the water quality maps produced in the SIMILE is a customised `GeoNode <https://geonode.org/>`_ project instance. GeoNode is an open resource geospatial content management system that facilitates the publication of geospatial data through a web application.
